easyspin

Information and tasks relating to the EasySpin installation.

Syntax
easyspin info
easyspin doc
easyspin compile
easyspin ?
easyspin
Description

easyspin info> displays information about your current EasySpin installation. See easyspininfo.

easyspin doc> opens the documentation of your current EasySpin installation. See easyspininfo.

easyspin compile> compiles the C files of your current EasySpin installation using mex. See easyspincompile.

easyspin ?> displays al possible options for easyspin.

easyspin> is equivalent to easyspin info.
